# About
- Tiny simple web scraper in Rust using the webpage Crate.
- Extracts the title, body, and any hyperlinks found within the specified URL.
- Planning to add more to this tiny project in the future.

# Usage
- The implementation allows for external usage
- cargo build & cargo run to use

# Extra
- This project is not complete, if you have any suggestions, submit a pr.
